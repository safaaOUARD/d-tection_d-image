# 🖼️ Image Detection App

## Description

Image Detection App est une application Android basée sur l'intelligence artificielle permettant d'identifier automatiquement une personne à partir d'une image. L'application utilise un modèle de reconnaissance d'images afin d'analyser les photos et déterminer à qui elles appartiennent.

## Fonctionnalités principales

### 📷 Capture d'image en temps réel

L'utilisateur peut prendre une photo directement depuis la caméra du téléphone pour effectuer une détection instantanée.

### 🖼️ Sélection d'une image depuis la galerie

L'application permet également de choisir une photo existante dans la galerie du téléphone afin de l'analyser.

### 🤖 Reconnaissance par Intelligence Artificielle

Une fois l'image sélectionnée ou capturée, un modèle d'intelligence artificielle analyse les caractéristiques visuelles de la photo.

### 👤 Identification de la personne

Après l'analyse, l'application affiche le nom de la personne reconnue lorsque celle-ci est présente dans la base de données du modèle.

### ⚡ Analyse rapide

Le traitement est effectué directement dans l'application afin d'obtenir un résultat rapide et fluide.

### 📱 Interface simple et intuitive

L'utilisateur peut facilement choisir entre la capture d'une nouvelle photo ou la sélection d'une image existante pour lancer la détection.

## Technologies utilisées

* Android Studio
* Java
* TensorFlow Lite
* Machine Learning
* Camera API
* Galerie Android
